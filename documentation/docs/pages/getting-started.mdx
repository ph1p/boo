# Getting Started

See [Installation](/installation) for download and build instructions.

## First Launch

1. Open Boo — it starts with a workspace in your home directory
2. **Cmd+Shift+O** to open a project folder as a workspace
3. **Cmd+B** to toggle the sidebar file explorer
4. **Cmd+D** / **Cmd+Shift+D** to split panes
5. **Cmd+,** to open settings and pick a theme

## Key Concepts

- **Workspaces** are project folders, each with its own color and state
- **Panes** are split views within a workspace, each with its own tab bar
- **Tabs** are terminal sessions within a pane
- **Sidebar** shows plugins (file tree, git, docker, bookmarks) for the active tab

Close order is smart: **Cmd+W** closes tab → pane → workspace.
