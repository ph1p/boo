# Installation

## Download

Download the latest release from [GitHub Releases](https://github.com/ph1p/boo/releases).

1. Download `Boo-x.x.x.dmg`
2. Open the DMG and drag **Boo** to **Applications**
3. Launch Boo

:::note
On first launch, macOS may show a security prompt. Click "Open" to allow it. Releases are signed and notarized.
:::

Boo checks for updates on launch. See [Updates](/updates).

## Build from Source

**Prerequisites:** macOS 13+, Xcode CLT, Zig 0.15+ (`brew install zig`), Rust (`rustup`), Metal Toolchain

```bash
git clone --recursive https://github.com/ph1p/boo.git
cd boo
make setup    # Builds GhosttyKit + ironmark + Boo
make run      # Build and launch
```

If you already cloned without `--recursive`:

```bash
git submodule update --init --depth 1
```

:::note
First build downloads dependencies and compiles Metal shaders. Takes a few minutes. Subsequent builds are cached.
:::

### Release Build

```bash
make release      # Optimized build
make app          # Create .app bundle
make dmg          # Create distributable DMG
```

### Tests

```bash
make test
```

## System Requirements

| Requirement       | Minimum               |
| ----------------- | --------------------- |
| macOS             | 13.0 (Ventura)        |
| Architecture      | Apple Silicon (arm64) |
| Disk space        | ~200MB                |
| Zig (build only)  | 0.15+                 |
| Rust (build only) | stable                |

## Uninstall

1. Quit Boo
2. Delete `Boo.app` from Applications
3. Optionally remove configuration: `rm -rf ~/.boo`
