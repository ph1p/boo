# Installation

Boo is currently documented and tested for macOS 13+ on Apple Silicon.

## Download

Download the latest release from the [GitHub Releases](https://github.com/ph1p/boo/releases) page.

### macOS (DMG)

1. Download `Boo-x.x.x.dmg`
2. Open the DMG and drag **Boo** to your **Applications** folder
3. Launch Boo from Applications

:::note
On first launch, macOS may show a security prompt. Click "Open" to allow the app to run. Releases are signed and notarized with Apple Developer ID.
:::

Once installed, Boo can check GitHub Releases on launch and update in place. See [Updates](/updates).

## Build from Source

**Prerequisites:** macOS 13+, Xcode CLT, Zig 0.15+, Rust, Metal Toolchain

```bash
git clone --recursive https://github.com/ph1p/boo.git
cd boo
make setup    # Builds GhosttyKit + Boo
make run      # Build and launch
```

### Step by Step

#### 1. Clone the repository

```bash
git clone --recursive https://github.com/ph1p/boo.git
cd boo
```

If you already cloned without `--recursive`, initialize the submodule:

```bash
git submodule update --init --depth 1
```

#### 2. Build

```bash
make setup
```

#### 3. Run

```bash
make run
```

### Release Build

```bash
make release      # Optimized build
make app          # macOS: create .app bundle
make dmg          # macOS: create distributable DMG
```

### Running Tests

```bash
make test     # Runs the macOS test suite
```

## System Requirements

| Requirement       | Minimum                       |
| ----------------- | ----------------------------- |
| macOS             | 13.0 (Ventura)                |
| Architecture      | Apple Silicon (arm64)         |
| Disk space        | ~200MB (including GhosttyKit) |
| Zig (build only)  | 0.15+                         |
| Rust (build only) | stable                        |

## Uninstall

1. Quit Boo
2. Delete `Boo.app` from Applications
3. Optionally remove configuration: `rm -rf ~/.boo`
